# Cloud-Edge Bayesian Optimization for Hyperparameter Tuning

## Research Project
Parallel Bayesian optimization framework for heterogeneous cloud-edge infrastructure.

## Novelty
- First parallel BO for heterogeneous edge workers
- q-UCB batch acquisition for cloud-edge orchestration
- Hardware-aware metrics (accuracy + runtime + CPU + memory)

## Tools
- Cloud: AWS EC2 (GP surrogate model)
- Training: Google Colab (CNN training)
- IDE: VS Code
- Containerization: Docker (edge worker simulation)
- Version Control: GitHub

## Datasets
- MNIST
- CIFAR-10
- Fashion-MNIST
